# 🎮💻 All-in-One Coding AI

Personal Coding Assistant for **Game Development** and **Website Development** using **Grok (xAI)** + **Gemini (Google)**.

## Features
- Supports every game engine and web framework
- Compare Grok 4.20 and Gemini 3.1 Pro side-by-side
- Clean, commented, ready-to-run code
- Beautiful Streamlit UI

## Setup
1. `pip install -r requirements.txt`
2. Copy `.env.example` to `.env` and add your API keys
3. `streamlit run coding_ai.py`

Get keys:
- Grok: https://console.grok.x.ai/
- Gemini: https://ai.google.dev/

Enjoy building games and websites!
